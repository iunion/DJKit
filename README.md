DJKit 
==============

DJKit is a collection of useful classes to **develop Apps faster**. <br/>

## Requirements

- iOS 4.0 or later
- Xcode 7.3 or later

## Frameworks

- libz

## Installation
#### CocoaPods

1. Add `pod 'DJKit'` to your Podfile.
2. Run `pod install` or `pod update`.
3. Import \<DJKit/DJKit.h\>.
